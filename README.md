# ChessServiceDemo
1. 使用SpringCloud架构模式实现微服务
2. 使用Eruake实现服务发现注册
3. 使用2个数据源：mysql , grpc +第三方接口数据，不同的数据源通过界面上不同的入口访问（不同的URI）
4. grpc使用Go语言实现
5. 后端服务实现多版本
